# Simple Task Manager

## 🌟 Overview
This is a simple web-based task manager application that allows users to add, view, update, and delete tasks efficiently.


## 🚀 Features
- Add new tasks
- View all tasks
- Edit tasks
- Delete tasks


## 🛠️ Technologies Used

### Frontend
- HTML
- CSS

### Backend
- Python (Flask)

### Database
- SQLite
